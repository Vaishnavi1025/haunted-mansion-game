# Haunted Mansion Text Adventure Game

A fun, text-based RPG written in Python where the player must navigate a haunted mansion to escape. 

## How to Play
1. Run the script in your terminal using `python game.py` (or your file name).
2. Enter your name to begin the adventure.
3. Explore different areas like the **Gate**, **Hall**, or **Garden** by typing your choices.
4. Find **3 keys** to unlock the main gate and win, while managing your health to survive ghost attacks!

## Features
* Interactive text-based choices.
* Randomized outcomes for exploration using Python's `random` module.
* Live player status tracking (Health and Keys).
